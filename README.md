# codex-dev-skill

Claude Code 플러그인. Codex에게 구현을 위임하고 Claude가 검수하는 `codex-dev` 스킬.

## 설치

```
/plugin marketplace add zoo3323/codex-dev-skill
/plugin install codex-dev@codex-dev
```

## 사용법

Claude Code에서 `/codex-dev` 실행. 첫 실행 시 Codex 권한 자동 허용 + 모델/이펙트 선택 질문이 뜬다.

## 업데이트

```
/plugin marketplace update codex-dev
```

## 제거

```
/plugin uninstall codex-dev@codex-dev
```
